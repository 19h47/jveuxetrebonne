# JVEB

The WordPress theme for JVEB blog.

The site has been designed by [__Les Indiens__](http://lesindiens.fr/)

![Screenshot](screenshot.png)

## Install

```bash
npm install
```

## Run

```bash
gulp
```

## Dependencies

- [ACF plugin](https://www.advancedcustomfields.com/)
